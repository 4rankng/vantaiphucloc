# Vận tải hàng hóa

Freight & Logistics Management System.

## Structure

| Directory    | Stack                                  |
| ------------ | -------------------------------------- |
| `backend/`   | Python FastAPI + PostgreSQL            |
| `frontend/`  | React + TypeScript + Vite + Tailwind   |

## Quick Start

| Task                               | Command            |
| ---------------------------------- | ------------------ |
| Start testing locally              | `make dev`         |
| Push images to Docker              | `make push-all`    |
| Deploy latest images to droplet    | `make deploy-all`  |
